 🏆 SportsNetwork

A production-ready full-stack sports news platform.
Live scores, player profiles, news articles, stadium info, and an AI-powered admin dashboard.

 🌐 Live Demo
👉 https://sportsnetwork.onrender.com

 🛠️ Tech Stack
- Frontend: React 19, TypeScript, Tailwind CSS, Vite
- Backend: Node.js, Express
- Database: MongoDB Atlas
- AI: Google Gemini AI
- Auth: JWT

✨ Features
- Live sports scores & news feed
- Player & stadium detail pages
- AI-powered content (Gemini)
- Admin dashboard with login
- Fully responsive design

🚀 Run Locally
1. Clone the repo
```bash
   git clone https://github.com/shivam23-tiwari/SportsNetwork.git
   cd SportsNetwork
```
2. Install dependencies
```bash
   npm install
```
3. Create `.env` file
MONGO_URI=https://cloud.mongodb.com/v2/69f2e8084546f383c7f908c1#/overview
NEWS_API_KEY=6d6bbfc288546249afda9b58ccd4eea

 4. Start the app
```bash
   npm run dev
```
